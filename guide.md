# Installation Guide

Follow these simple steps to install the **Upwork Bid Checker** extension on your Chrome browser.

## 1. Prerequisites
- You need the **Google Chrome** browser installed on your computer.
- You need the extension files (downloaded from this repository).

## 2. Prepare the Files
1. Download this repository to your computer (usually as a ZIP file).
2. **Unzip** (extract) the folder to a location you can easily access (e.g., your Desktop or Documents).
3. You should see a folder named `extension` inside. This is what we will load.

## 3. Load into Chrome
1. Open Google Chrome.
2. In the address bar, type `chrome://extensions` and press **Enter**.
3. In the top-right corner, look for a switch that says **Developer mode**. Turn it **ON**.
4. A new menu bar will appear. Click the button that says **Load unpacked**.
5. A file browser window will open. Navigate to the folder where you unzipped the files.
6. Select the `extension` folder and click **Select Folder** (or Open).

## 4. Pin the Extension
1. You should now see the "Upwork Bid Checker" in your list of extensions.
2. Click the **Puzzle Piece icon** üß© in the top-right corner of your Chrome browser (next to the address bar).
3. Find "Upwork Bid Checker" in the list and click the **Pin icon** üìå next to it.
4. The extension icon should now be visible in your toolbar.

## 5. How to Use
1. Go to any **Upwork Job Post** page.
2. Click the extension icon in your toolbar.
3. Enter your **Name** (you only need to do this once).
4. Click **Save URL**.
   - If the job is new, it will say **Saved ‚úÖ**.
   - If someone else already saved it, it will say **Already exists ‚ùå**.
5. To see recent bids, click the **View Bids** button (or the icon in the popup).
